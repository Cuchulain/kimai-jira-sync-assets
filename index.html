Quick start

Install fast, configure once, keep working normally.

  • Download the latest public ZIP from GitHub and drop it into your Kimai plugins directory.
  • Run the install command, activate the license, and set your Jira project details in the Kimai UI.
  • Import tasks and start syncing worklogs with the issue keys your team already uses.
